Source code | Home page / Source code simplifier

General rules:

  • Classes are childs of the module
  • Compiler errors are not fixed
  • Defs are childs of the module or of classes
  • Everything that can be sorted is sorted
  • Forbid/replace anything complex when something newer/simpler is available
  • Forbid/remove anything that does not contribute to the result
  • Only one module named main.py

Installation:

python3 -m pip install https://github.com/pbizopoulos/source-code-simplifier/archive/main.zip

Usage:

source_code_simplifier code_input.py > code_output.py
