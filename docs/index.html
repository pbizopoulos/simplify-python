General rules:

  • There is one and only one module named main.py (local imports are forbidden)
  • The module has only one argument: DEBUG (argparse import is forbidden)
  • The module is passed through the Python ast parser and unparser
  • The children of the module are placed in the following order:
    1. from import statements
    2. import statements
    3. class statements
    4. def statements
    5. if __name__ == '__main__': main()
  • Classes only contain defs
  • Sort:
    1. from import statements according to package name
    2. import statements according to package name
    3. classes according to name
    4. defs according to name
    5. defs arguments according to name
  • Forbid/replace anything old/complex when something newer/simpler is available
  • Forbid/remove anything that does not contribute to the result (e.g. print statements)
    1. logging import
    2. print statements

Installation:

python3 -m pip install https://github.com/pbizopoulos/source-code-simplifier/archive/main.zip

Usage:

source_code_simplifier code_input.py
